# Sudoku

Fill the 9×9 grid so that every row, column, and 3×3 box contains the digits 1–9 exactly once. Puzzles are procedurally generated at three difficulty levels.

---

## Options

Configure before starting:

| Option | Values |
|--------|--------|
| Difficulty | Easy / Medium / Hard |
| Highlight color | Blue / Green / Red / Gray / None |

The highlight color marks cells that share the same digit as the cursor.

---

## Controls

| Key | Action |
|-----|--------|
| `↑ ↓ ← →` | Move cursor |
| `1`–`9` | Enter digit (or pencil mark in pencil mode) |
| `Space` | Toggle pencil mode |
| `Backspace` / `Delete` | Clear current cell |
| `U` | Undo (up to 100 moves) |
| `H` | Toggle matching-number highlight |
| `P` | Pause |
| `Esc` | Quit to menu (press twice to confirm) |

---

## Pencil Marks

Press `Space` to enter pencil mode. In pencil mode, digits toggle candidate marks on a cell rather than placing a value. Pencil marks are cleared when you place a digit normally.

---

## Scoring

Score starts at **10 000** and counts down as time passes. The elapsed time is shown in the HH:MM:SS format. Complete the puzzle as fast as possible for the highest score.
