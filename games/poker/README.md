# Poker

Texas Hold'em — bet, raise, or fold.

---

## Options

| Option | Values | Description |
|--------|--------|-------------|
| Seats | 2–6 | Number of AI opponents at the table |
| Difficulty | Easy, Medium, Hard | AI opponent skill level |

---

## Controls

| Key | Action |
|-----|--------|
| `F` | Fold — forfeit your hand |
| `C` | Call — match the current bet, or check if no bet |
| `R` | Raise — enter raise mode |
| `A` | All-in — bet all your chips |
| `↑` / `↓` | Adjust raise amount (in raise mode) |
| `Enter` | Confirm raise (in raise mode) |
| `Esc` | Cancel raise (in raise mode) |

---

## Hand Rankings

| Rank | Hand | Description |
|------|------|-------------|
| 1 | Royal Flush | A, K, Q, J, 10 all of the same suit |
| 2 | Straight Flush | Five consecutive cards of the same suit |
| 3 | Four of a Kind | Four cards of the same rank |
| 4 | Full House | Three of a kind plus a pair |
| 5 | Flush | Five cards of the same suit |
| 6 | Straight | Five consecutive cards of mixed suits |
| 7 | Three of a Kind | Three cards of the same rank |
| 8 | Two Pair | Two different pairs |
| 9 | One Pair | Two cards of the same rank |
| 10 | High Card | Highest card when no other hand is made |

---

## Rules

- **Blinds:** Small blind (10 chips) and big blind (20 chips) are posted before the flop. Action starts left of the big blind.
- **Phases:** Preflop → Flop (3 cards) → Turn (4th card) → River (5th card) → Showdown
- **Betting:** Players can fold, call, raise, or go all-in. Minimum raise is 20 chips.
- **Win Condition:** Best five-card hand wins the pot. If only one player remains (all others folded), that player wins automatically.
