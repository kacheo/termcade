# Tetris

Classic block-stacking game with all standard mechanics including hold piece, ghost piece, and configurable start level.

---

## Options

Configure before starting:

| Option | Values | Default |
|--------|--------|---------|
| Ghost piece | On / Off | Off |
| Start level | 0–9 | 0 |

Higher start level = faster initial drop speed.

---

## Controls

| Key | Action |
|-----|--------|
| `← →` | Move |
| `↑` | Rotate |
| `↓` | Soft drop |
| `Space` | Hard drop |
| `P` | Pause / Resume |
| `Q` | Quit to menu |

---

## Scoring

| Lines cleared | Points |
|---------------|--------|
| 1 | 100 × (level + 1) |
| 2 | 300 × (level + 1) |
| 3 | 500 × (level + 1) |
| 4 (Tetris!) | 800 × (level + 1) |

Soft drop: +1 pt/row · Hard drop: +2 pt/row · Level up every 10 lines cleared.
